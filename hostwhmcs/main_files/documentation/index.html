HTML STRUCTURE

The "HostWHMCS" template is a responsive HTML template based on Bootstrap 3.3.7. The structure based on a flexible grid that lets you change HTML pages quickly and logically with a nestable system.

If you would like to edit the layout of any elements in whole template, you would do the following:

Open any '.html' or, '.css', so that is extremely easy to change any html markup and css attributes. Simply find the related and commented heading in the html or, css file and make your magic.

FAQ

How can I remove the "Preloader"?

  • Open Up your Desired '.html' file. (For Example, 'index.html')
  • Then Search For the Comment '<!-- Preloader Start -->'
  • And remove the markup



How can I change the "Logo"?

  • Open Up your Desired '.html' file. (For Example, 'index.html')
  • Then Search For the Comment '<!-- Logo Start -->'
  • Update The logo name 'Eco Hosting' from there.



How can I change "Slider Text"?

  • Open Up your Desired '.html' file. (For Example, 'index.html')
  • Then Search For the Comments '<!-- Banner Content Start -->'
  • Update The 'content' from there



How can I change the "Slider Background Image"?

  • Open Up your Desired '.html' file. (For Example, 'index.html')
  • Then Search For the Comment '<!-- Banner Area Start -->'
  • Update The 'data-bg-img="img/banner-slider-img/rocket.png"' from there

How can I change the "Mailchimp Subscription Link" in HTML and WHMCS Template both?
