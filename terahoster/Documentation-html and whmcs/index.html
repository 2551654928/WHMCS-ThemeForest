6.1 WHMCS THEME Costumization

We have made very easy to costumize WHMCS Theme.You need to edit only two files : Header.tpl and Footer.tpl (the edit procedure is the same as html pages)

Follow these simple steps :

Open TeraHoster.zip > whmcs-theme > Inside of WHMCS Theme folder search for : header.tpl and footer.tpl
once you find those files,you have to open with one of your text editors (notepad,notepad++,brackets etc),same as HTML pages.

6.1.2 CSS File of WHMCS

Inside WHMCS Theme you will see a folder called "css",open that folder and you will se a lot of CSS Files.To make changes to TeraHoster you need to open "New-Style.css" . These are the files that you need to use if you want to make custom design for WHMCS Theme.
