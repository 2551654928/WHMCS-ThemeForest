Installing HostLab

1 - Unzip the download pack, view template file in any browser, you can display or edit without an internet connection

2 - Open your FTP Client (FileZilla for example), and upload the content of the template folder on your server root;

3 - Finaly and once the files are done uploading go to www.yourdomain.com/index.html

Installing WHMCS

1 - Unzip the download pack "HostLab", inside WHMCS folder.

2 - Open your FTP Client (FileZilla for example), and upload the "HostLab" folder on WHMCS root directory inside the "Template" folder.

3 - Now go to WHMCS administrator panel.

4 - Click on Setup -> General Settings. Set "Template" value to "HostLab".

Note: Theme bundle comprises whmcs folder containing templates folder and some PHP files. Upload both templates folder as well as PHP files at the WHMCS root directory. Overwrite all.

Feel free to contact via email about installing WHMCS.
