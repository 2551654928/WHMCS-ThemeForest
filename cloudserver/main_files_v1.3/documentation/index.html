INTRODUCTION

CloudServer is a Responsive HTML5 Technology, Web Hosting and WHMCS Template designed for All kinds of Technology, Domain and Hosting Business. It's an HTML5 template based on latest Bootstrap 3.3.7. Also included 10 WHMCS Template based on WHMCS v7.2. Anyone can easily update/edit this plugin to follow our Well Sorted Documentation.
Mainly, our template is an HTML template and we just provided support by customizing WHMCS header and footer and color based on our template for FREE of cost with our HTML template.

HTML STRUCTURE

The "CloudServer" template is a responsive HTML template based on Bootstrap 3.3.7. The structure based on a flexible grid that lets you change HTML pages quickly and logically with a nestable system.

If you would like to edit the layout of any elements in whole template, you would do the following:

Open any '.html' or, '.css', so that is extremely easy to change any html markup and css attributes. Simply find the related and commented heading in the html or, css file and make your magic.

INSTALLATION

Via FTP:

1. Open up your 'FTP Software' of choice and connect to your hosting

2. Browse to required directory (Normally: ./public_html)

3. Unzip the 'main_files.zip' and choose which templates to upload.

4. Upload the template to the server.

CUSTOMIZATION

Once you finished setup successfully, you can customize it as you want. Below are some common customization questions.

Q. How To Remove Preloader?

  • Open '.html' file. (For Example: 'index.html')
  • Find the comment 'FakeLoader Start'
  • Then the remove code block

Q. How To Change Promo Bar Text and Date/Time?

  • Open '.html' file. (For Example: 'index.html')
  • Find the comment 'Promo Area Start'
  • Then update the date

Q. How To Change Topbar Text and Social Links?

  • Open '.html' file. (For Example: 'index.html')
  • Find the comment 'Primary Menu Start'
  • Then update text and links

Q. How To Change Logo?

  • Open '.html' file. (For Example: 'index.html')
  • Find the comment 'Logo Start'
  • Update the 'src' value of img tag

Q. How To Change Off-Canvas Logo?

  • Open '.html' file. (For Example: 'index.html')
  • Find the comment 'Off-Canvas Menu Logo Start'
  • Update the 'src' value of img tag

Q. How To Change Off-Canvas Menu Links?

  • Open '.html' file. (For Example: 'index.html')
  • Find the comment 'Off-Canvas Menu Links Start'
  • Update the lists and links

Q. How To Change Banner Slider Text?

  • Open '.html' file. (For Example: 'index.html')
  • Find the comment 'Banner Item Content Start'
  • Update the contents

Q. How To Change Banner Slider Image?

  • Open '.html' file. (For Example: 'index.html')
  • Find the comment 'Banner Item Image Start'
  • Update the location of the img tag

Q. How To Change Banner Slider Background Image?

  • Open '.html' file. (For Example: 'index.html')
  • Find the comment 'Banner Item Start'
  • Update the image location of 'data-bg-img' attribute

Q. How To Change Banner Slider Background Video?

  • Open '.html' file. (For Example: 'index.html')
  • Find the comment 'Banner Item Start'
  • Update location of 'data-bg-video' attribute with any 'youtube video id'

Q. How To Change Banner Slider Nav?

  • Open '.html' file. (For Example: 'index.html')

Q. How To Change Pricing Contents?

Q. How To Change Counter Item Text and Icon?
